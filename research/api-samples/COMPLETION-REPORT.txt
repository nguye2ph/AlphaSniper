TECHNICAL SPECIFICATIONS
================================================================================

SEC EDGAR:
  Endpoint: https://data.sec.gov/submissions/CIK{cik}.json
  Method: GET
  Auth: None required
  Rate Limit: 10 requests/second
  Response Format: JSON
  Response Size: 150-200 KB per company
  Update Frequency: < 1 second

Finnhub:
  Base: https://finnhub.io/api/v1/
  Auth: API key in query parameter (token=)
  Rate Limit: 60 requests/minute
  Response Format: JSON
  Data Types: News, quotes, insider transactions, sentiment
  Signup: https://finnhub.io/register

MarketAux:
  Base: https://api.marketaux.com/v1/
  Auth: API token in query parameter (api_token=)
  Rate Limit: 60 requests/minute
  Response Format: JSON
  Data Types: News, entities, sentiment
  Signup: https://www.marketaux.com/
